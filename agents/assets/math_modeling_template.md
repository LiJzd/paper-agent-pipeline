# 数学建模论文模板（基于12篇国赛优秀论文提炼）

> 本模板基于全国大学生数学建模竞赛12篇优秀论文（A/B/C/D题各3篇）的格式与表达模式提炼而成，适用于国赛D题及其他赛题。

---

## 一、封面格式

```
队伍编号 MC2500XXXX
题号 D

[论文标题]

摘 要
[摘要正文，300-500字]

关键词：[关键词1]；[关键词2]；[关键词3]；[关键词4]；[关键词5]
```

### 论文标题规范
- 标题应体现**研究对象 + 核心方法/模型**，如：
  - "基于XGBoost与多阶段联合调度的货量预测与车辆优化模型"
  - "基于多目标优化的老城搬迁智能决策模型"
  - "基于随机森林的音频信号分类与去噪方法"
- 避免过于宽泛的标题，如"XX问题的研究"

## 二、摘要写作规范（占评分20%，最重要）

### 整体结构（300-500字）

```
[开头段：背景+目的，1-2句]
针对[问题领域]中的[具体问题]，本文构建了[模型类型]，实现了[目标]。

[问题一段：针对问题一]
针对问题一，本文基于[数据/方法]，采用[具体方法]，建立[模型名称]。
通过[关键步骤]，实现了[目标]，得到[具体结果]（如RMSE=0.9751，R²=0.9322）。

[问题二段：针对问题二]
针对问题二，本文在问题一的基础上，[方法描述]。
[求解过程]，得到[具体结果]（如搬迁成功率达到95.58%）。

[问题三段...]
...

[结尾段：总结价值]
综上所述，本文构建的[模型名称]具有[优点]，能够为[应用领域]提供理论支持与实践指导。
```

### 摘要写作要点

1. **必须包含具体数据**：每个问题段必须有量化结果（RMSE、R²、准确率、成功率等）
2. **方法名称要具体**：写"XGBoost回归模型"而非"机器学习模型"
3. **问题段逻辑**：问题是什么 → 用了什么方法 → 怎么做的 → 结果是多少
4. **避免空洞表述**：不要写"取得了较好效果"，要写"R²达到0.9322"
5. **关键词5-8个**：涵盖研究对象、核心方法、应用场景

### 摘要示例（综合多篇优秀论文）

```
针对短途运输中的货量预测与车辆调度优化问题，本文综合运用时间序列分析、
机器学习和运筹优化方法，构建了多阶段联合调度模型。

针对问题一，本文基于附件提供的历史货量数据，采用XGBoost回归算法建立
货量预测模型。通过提取时间特征、周期特征和滞后特征，经5折交叉验证调参，
模型在测试集上的RMSE为0.9751，R²为0.9322，预测精度优于ARIMA和LSTM基线。

针对问题二，本文在问题一预测结果的基础上，建立以最小化运输成本为目标的
整数规划模型。考虑车辆容量、时间窗、装载率等约束，采用改进遗传算法求解，
得到最优调度方案，总运输成本降低18.3%，装载率提升至87.5%。

针对问题三，本文引入多目标优化框架，同时优化运输成本和客户满意度。
采用NSGA-II算法求解Pareto前沿，得到一组非支配解供决策者选择。

综上所述，本文构建的多阶段联合调度模型具有预测精度高、求解效率好、
实用性强等优点，能够为短途运输企业的运营决策提供科学依据。

关键词：短途运输；货量预测；XGBoost；整数规划；多目标优化

```

**写作要点：**

## 五、公式写作规范

### 公式格式
- **行内公式**：使用 $...$ 标记
- **行间公式**：独立成行，右对齐编号
- **编号格式**：(1)、(2)、(3)...，右对齐
- **首次引用**：每个公式首次出现时必须有编号

### 常见公式模板

#### 预测类公式
```
# 均方根误差
RMSE = √(1/n · Σᵢ₌₁ⁿ (yᵢ - ŷᵢ)²)    (1)

# 决定系数
R² = 1 - Σᵢ₌₁ⁿ (yᵢ - ŷᵢ)² / Σᵢ₌₁ⁿ (yᵢ - ȳ)²    (2)

# 平均绝对误差
MAE = 1/n · Σᵢ₌₁ⁿ |yᵢ - ŷᵢ|    (3)

# 平均绝对百分比误差
MAPE = 1/n · Σᵢ₌₁ⁿ |yᵢ - ŷᵢ| / yᵢ × 100%    (4)
```

#### 优化类公式
```
# 目标函数（最小化总成本）
min Z = Σᵢ₌₁ᴺ Σⱼ₌₁ᴹ cⱼ · xᵢⱼ · dᵢ    (5)

# 约束条件（任务分配唯一性）
Σⱼ₌₁ᴹ xᵢⱼ = 1,  ∀i ∈ {1,2,...,N}    (6)

# 约束条件（资源容量限制）
Σᵢ₌₁ᴺ dᵢ · xᵢⱼ ≤ Qⱼ,  ∀j ∈ {1,2,...,M}    (7)

# 决策变量定义
xᵢⱼ ∈ {0,1},  ∀i,j    (8)
```

#### 评价类公式
```
# 层次分析法一致性指标
CI = (λmax - n) / (n - 1)    (9)

# 一致性比率
CR = CI / RI    (10)

# TOPSIS综合得分
Sᵢ = Dᵢ⁻ / (Dᵢ⁺ + Dᵢ⁻)    (11)
```

#### 分类类公式
```
# 准确率
Accuracy = (TP + TN) / (TP + TN + FP + FN)    (12)

# 召回率
Recall = TP / (TP + FN)    (13)

# F1值
F1 = 2 · Precision · Recall / (Precision + Recall)    (14)
```
