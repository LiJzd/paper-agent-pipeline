## 数学建模论文写作规范

### 一、摘要写作要求（占评分20%，最重要）

**结构：**
```
[开头段：背景+目的，1-2句]
[问题一段：方法+过程+量化结果]
[问题二段：方法+过程+量化结果]
...
[结尾段：总结优点+实用价值]
```

**模板：**
```
针对[问题领域]中的[具体问题]，本文构建了[模型类型]，实现了[目标]。

针对问题一，本文基于[数据/方法]，采用[具体方法]，建立[模型名称]。
通过[关键步骤]，实现了[目标]，得到[具体结果]（如RMSE=0.9751，R²=0.9322）。

针对问题二，本文在问题一的基础上，[方法描述]。
[求解过程]，得到[具体结果]。

综上所述，本文构建的[模型名称]具有[优点]，能够为[应用领域]提供理论支持与实践指导。
```

**强制要求：**
- 每个问题段必须包含量化结果（具体数字）
- 方法名称要具体（"XGBoost"而非"机器学习"）
- 300-500字
- 关键词5-8个

### 二、问题重述写作要求

#### 1.1 问题背景
- 字数：200-300字
- 内容：行业背景、政策背景、问题重要性
- 语气：客观、简洁、学术化

**模板：**
```

```

#### 1.2 需要解决的问题

## 公式写作规范

### 公式格式
- 行内公式：$...$
- 行间公式：独立成行，右对齐编号
- 编号格式：(1)、(2)、(3)...，右对齐

### 常见公式模板

#### 预测类
```
RMSE = √(1/n · Σᵢ₌₁ⁿ (yᵢ - ŷᵢ)²)    (1)
R² = 1 - Σᵢ₌₁ⁿ (yᵢ - ŷᵢ)² / Σᵢ₌₁ⁿ (yᵢ - ȳ)²    (2)
MAE = 1/n · Σᵢ₌₁ⁿ |yᵢ - ŷᵢ|    (3)
MAPE = 1/n · Σᵢ₌₁ⁿ |yᵢ - ŷᵢ| / yᵢ × 100%    (4)
```

#### 优化类
```
min Z = Σᵢ Σⱼ cⱼ · xᵢⱼ · dᵢ    (1)
Σⱼ xᵢⱼ = 1, ∀i    (2)
Σᵢ dᵢ · xᵢⱼ ≤ Qⱼ, ∀j    (3)
xᵢⱼ ∈ {0,1}, ∀i,j    (4)
```

#### 评价类
```
CI = (λmax - n) / (n - 1)    (1)
CR = CI / RI    (2)
Sᵢ = Dᵢ⁻ / (Dᵢ⁺ + Dᵢ⁻)    (3)
```

## 图表生成规范

### 图表类型
1. **数据可视化**：matplotlib/seaborn
2. **流程图**：matplotlib/networkx
3. **结果图**：matplotlib/plotly
4. **分析图**：matplotlib/seaborn

### 图表代码模板
```python
import matplotlib.pyplot as plt
import numpy as np

plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei']
plt.rcParams['axes.unicode_minus'] = False

fig, ax = plt.subplots(figsize=(10, 6))
ax.plot(x, y, label='标签')
ax.set_xlabel('X轴标签')
ax.set_ylabel('Y轴标签')

ax.grid(True, alpha=0.3)
```

- 图：`图X [标题]`（标题在图片**下方**，居中）

## 格式强制要求

### 引用上标拆分
```python
# 引用 [N] 作为独立 run，设置 superscript
import re
def add_paragraph_with_citations(doc, text):
    parts = re.split(r'(\[\d+(?:[,，\s]*\d+)*(?:\s*[-–—]\s*\d+)?\])', text)
    para = doc.add_paragraph()
    for part in parts:
        if re.match(r'\[\d+(?:[,，\s]*\d+)*(?:\s*[-–—]\s*\d+)?\]', part):
            run = para.add_run(part)
            run.font.superscript = True
        else:
            para.add_run(part)
```
